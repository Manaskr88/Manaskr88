## About

I am **Manas Kumar**, a pre-final year **B.Tech Computer Science Engineering** student at **Galgotias University**, focused on building scalable, secure, and product-driven software systems.

My engineering interests sit at the intersection of **full stack development**, **AI-enabled applications**, **software architecture**, and **problem solving**. I enjoy transforming ideas into reliable digital products using modern web technologies, clean backend systems, intuitive interfaces, and efficient engineering practices.

I have hands-on experience developing responsive full-stack applications using **React.js, Node.js, Express.js, MongoDB, Firebase, REST APIs, JWT authentication**, and modern frontend tooling. Alongside development, I actively strengthen my foundation in **Data Structures and Algorithms using Java**, system design fundamentals, and production-grade software workflows.

My long-term goal is to grow as a software engineer who can build high-impact products, contribute to engineering teams, and solve real-world problems through scalable technology.
